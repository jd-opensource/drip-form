## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}
{
      "type": "string",
      "title": "日期选择框",
      "format": "date-time",
      "ui": {
        "type": "datePicker",
        "range": true,
        "allowClear": true,
        "disabled": false,
        "rangeplaceholder": [
          {
            "startTime": "11:11:11"
          }
        ]
      },
      "fieldKey": "datePicker_tHj963"
    }
```

## API
| 参数        | 说明                             | 类型     |  是否必填 | 默认值 |
|:-----------|:---------------------------------|:--------|:---|:---|
| filedKey   | 表单数据存放字段（必须唯一）         | string   | 必填 | - |
| disabled   | 值为 `true` 时，日期选择框为禁用状态 | boolean  | 非必填 | false |
|format      | 时间格式                          | string   | 非必填 | `'YYYY-MM-DD HH:mm:ss'`|
| onChange   | 时间发生变化的回调                  | function(date: moment, dateString: string) => void   |  非必填 | - |
| picker     | 设置选择器类型                     |`time`、`date`、 `week`、 `month`、 `quarter`、` year` | 非必填| date|
|range       | 是否为范围选择器                   | boolean   | 非必填 | false |
| showTime   | 是否开启时间选择功能,如使用，`format`格式应包括`HH:mm:ss`| `boolean`  | 非必填| false |
| style      | 自定义输入框样式                   | React.CSSProperties[ ] | 非必填 | { } |
