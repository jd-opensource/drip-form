## 配置
该字段配置在`schema`中：

```json title="schema" {12,13,14}

{
    "validateTime": "submit",
    "type": "number",
    "title": "滑动输入条",
    "ui": {
        "range": false,
        "type": "slider",
        "marks": {
            "25": "quarter",
            "50": "middle",
            "75": "three-quarter"
        },
        "showNumber": true,
        "vertical": false,
        "step": 20,
        "dots": true,
        "min": 0,
        "max": 100
    },
    "fieldKey": "slider_WmZbgF"
}

```

## API
| 参数        | 说明     | 类型     |  是否必填 | 默认值 |
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| filedKey      | 表单数据存放字段（必须唯一）| string   | 必填 | - |
| disabled      | 值为 true 时，滑块为禁用状态 | boolean   | 非必填 | false |
| dots      | 是否只能拖拽到刻度上 | boolean   | 非必填 | false |
| marks      | 刻度标记，key 的类型必须为 number 且取值在闭区间 [min, max] 内，每个标签可以单独设置样式 | object   | 非必填 | { number: ReactNode } or { number: { style: CSSProperties, label: ReactNode } } |
|max |最大值 |number| 非必填 | 100 |
|min |最小值 |number| 非必填 | 0 |
| onChange      | 当 Slider 的值发生改变时，会触发 onChange 事件，并把改变后的值作为参数传入 | (value) => void   |  非必填 | - |
| placeholder      | 输入框占位文本 | string   | 非必填 | Please Select|
| range | 双滑块模式 | `boolean` 、 `range` | 非必填 | false |
| showNumber      | 当值为true时展示右侧的数字输入组件 | boolean | 非必填 | false |
| step      | 步长，取值必须大于 0，并且可被 (max - min) 整除。当 marks 不为空对象时，可以设置 step 为 null，此时 Slider 的可选值仅有 marks 标出来的部分 | `number` 、 `null`   | 非必填 | 1 |
| vertical      | 值为 true 时，Slider 为垂直方向 | boolean   | 非必填 | false |

### range
| 参数 | 说明| 类型 |是否必填 |默认值|
|:-----------|:--------------------------------------------------------------|:--------|:---|:---|
| draggableTrack | 范围刻度是否可被拖拽 | boolean | 非必填 | false |
