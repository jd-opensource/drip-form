TreeSelect 树选择

配置​

该字段配置在schema中：

schema
{
      "validateTime": "submit",
      "type": "array",
      "title": "树选择",
      "ui": {
        "type": "treeSelect",
        "requestCache": true,
        "showSearch": true,
        "multiple": true,
        "placeholder": "请选择您想要的选项",
        "options": [
          {
            "label": "shanghai",
            "value": "shanghai",
            "children": [
              {
                "label": "baoshan",
                "value": "baoshan"
              },
              {
                "label": "jingan",
                "value": "jingan"
              }
            ]
          },
          {
            "label": "beijing",
            "value": "beijing",
            "children": [
              {
                "label": "yizhuang",
                "value": "yizhuang"
              },
              {
                "value": "beichen",
                "label": "beichen"
              }
            ]
          }
        ]
      },
      "fieldKey": "treeSelect_iOBbGg"
    }
提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

API​

参数说明类型是否必填默认值
filedKey表单数据存放字段（必须唯一）string必填-
options可选数据源options[ ]必填-
allowClear是否允许清除boolean非必填false
disabled值为 true 时，选择器为禁用状态boolean非必填false
multiple支持多选（当设置 treeCheckable 时自动变为 true）boolean非必填false
onChange选中树节点时调用此函数function(value, label, extra)非必填-
placeholder占位文本string非必填-
requestCache是否请求缓存数据boolean非必填false
showCheckedStrategy配置 treeCheckable 时，定义选中项回填的方式。TreeSelect.SHOW_ALL: 显示所有选中节点(包括父节点)。TreeSelect.SHOW_PARENT: 只显示父节点(当父节点下所有子节点都选中时)。 默认只显示子节点all、parent、child非必填child
showSearch是否支持搜索框boolean非必填false
treeCheckable显示 Checkboxboolean非必填false
treeDataSimpleMode是否使用简单格式的treeData 即扁平化optionsboolean非必填false
withPidDelimiter包括pId的分隔符 即扁平化optionsstring非必填false
