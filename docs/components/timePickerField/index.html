TimePicker 时间选择器

配置​

该字段配置在schema中：

schema
{
      "type": "array",
      "title": "时间选择框",
      "ui": {
        "type": "timePicker",
        "range": true,
        "use12Hours": true,
        "showNow": true,
        "disabled": true
      },
      "fieldKey": "timePicker_YTyXPD"
    }
提示

组件具体配置依赖于所使用的UI库，DripForm仅作示例，具体可参考UI库对应文档。

API​

参数说明类型是否必填默认值
filedKey表单数据存放字段（必须唯一）string必填-
allowClear可以点击清除图标删除内容boolean必填true
disabled值为 true 时，时间选择器为禁用状态boolean非必填false
format时间格式string非必填HH:mm:ss
onChange时间发生变化的回调function(time: moment, timeString: string): void非必填-
placeholder占位文本string非必填请选择时间
range是否为范围选择器boolean非必填false
showNow面板是否显示“此刻”按钮boolean非必填false
use12Hours使用 12 小时制，为 true 时 format 默认为 HH:mm:ss aboolean非必填false
